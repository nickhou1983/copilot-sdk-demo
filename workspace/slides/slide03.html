技术栈

Backend 后端

Runtime: Node.js (v18+)

Framework: Express.js

实时通信: Socket.IO

语言: TypeScript 5.3

核心 SDK: @github/copilot-sdk v0.1.20

验证: Zod v3.22

Build 构建

TypeScript Compiler (tsc)

tsx 开发服务器 (watch mode)

ES2022 模块输出

Frontend 前端

语言: Vanilla JavaScript（无框架）

样式: 原生 CSS + CSS Variables

文件处理: HTML5 File API / 拖拽

代码高亮: Highlight.js (CDN)

核心依赖

@github/copilot-sdk

express

socket.io

multer

zod
